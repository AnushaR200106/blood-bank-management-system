# Blood Bank Management System

A web-based Blood Bank Management System developed using HTML, CSS, JavaScript, SQL, and DBMS concepts to manage donor information and blood inventory efficiently.

## Features
- Add and manage donor records
- Store blood group information
- Track blood inventory
- Search donor details
- User-friendly interface

## Technologies Used
- HTML
- CSS
- JavaScript
- SQL
- DBMS

## Project Structure
- index.html – Main webpage
- style.css – Styling and layout
- script.js – Client-side functionality
- database.sql – Database schema and queries

## Purpose
This project was developed to simplify blood bank management by organizing donor data and blood stock information in a structured database system.

- User authentication
