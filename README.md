# 🩸 Blood Bank Management System

A web-based Blood Bank Management System built using **HTML, CSS, and JavaScript** to manage donor registrations, patient blood requests, and blood inventory — all in a single responsive interface.

## 🌐 Live Demo
[Click here to view the live project](https://AnushaR200106.github.io/blood-bank-management-system)

---

## ✨ Features

- **Donor Registration** — Register donors with name, age, gender, blood group, phone, and city
- **Patient Blood Request** — Submit blood requests with hospital details and required units
- **Smart Status Logic** — Automatically marks requests as "Fulfilled" or "Pending" based on donor availability
- **Search & Filter** — Search donors by name or city, filter by blood group
- **Live Dashboard** — Real-time stats showing total donors, requests, and fulfilled count
- **Input Validation** — Phone number and age validation on all forms
- **Responsive Design** — Works on both desktop and mobile screens
